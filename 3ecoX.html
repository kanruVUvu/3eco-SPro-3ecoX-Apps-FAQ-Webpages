3ecoX Services FAQ

How‌ ‌to‌ ‌request‌ ‌a‌ Scheduled ‌Single Delivery?‌

  • Merchant has to enter the Pick up Address for the parcel. This data can either be fetched automatically from mobile GPS or can be entered manually.
  • Apart from the location, Merchant needs to specify the mobile number and Name of the person from whom the parcel needs to be picked up.
  • Merchant is prompted to enter the destination point. This can be fetched automatically from mobile GPS or can be input manually by Merchant .
  • Merchant is now prompted to enter the delivery address for the Package. This has to be input manually. Here also the delivery SPOC name and mobile number has to be entered.
  • Merchant has to choose whether the Delivery Type is Normal or Freight.
  • If the Delivery Type is Normal then the Merchant has to choose a multiple Seat Vehicle .
  • If the Delivery Type is Freight then the Merchant has to choose a Cargo Vehicle .
  • Merchant is now prompted to enter the Date for a Scheduled Delivery.This Date for a Scheduled Delivery is either equal to or greater than the Current Date .
  • Merchant is now prompted to enter the Time for a Scheduled Delivery.This time for a Scheduled Delivery is either 15 Minutes or more compared to the Current Time .
  • 3ecoX also warrants Merchant to categorize the parcel type. Also there is an option to upload parcel type File of maximum 5MB capacity. After which the Booking gets confirmed .

Can I see my Upcoming Deliveries in 3ecoX App?

Yes, To view your upcoming deliveries please follow the instructions below: Deliveries > My Deliveries > Upcoming

Can I see my Ongoing Deliveries in 3ecoX App?

Yes, To view your Ongoing deliveries please follow the instructions below: Deliveries > My Deliveries > Ongoing

Can I see my Completed Deliveries in 3ecoX App?

Yes, To view your completed deliveries please follow the instructions below: Deliveries > My Deliveries > Completed

Can I see my Cancelled Deliveries in 3ecoX App?

Yes, To view your cancelled deliveries please follow the instructions below: Deliveries > My Deliveries > Cancelled

Can I cancel a Scheduled Single / Multiple Delivery Request after Booking Confirmation?

Yes, To cancel a Scheduled Single / Multiple Delivery Request please follow the instructions below: Deliveries > Scheduled Delivery > Cancel Delivery Button to be clicked which is present in Scheduled Delivery Order

Can I cancel a Recurring Single / Multiple Delivery Request after Booking Confirmation?

Yes, To cancel a Recurring Single / Multiple Delivery Request please follow the instructions below: Deliveries > Recurring Delivery > Cancel Delivery Button to be clicked which is present in Recurring Delivery Order
